ex_camera_stop_fade

Stops all fade effects for the camera

Syntax

ex_camera_stop_fade(cameraHandle)
Argument Description
cameraHandle The camera handle
Returns nothing

Description

This function will stop any fade effects currently playing from the particular camera.

Example

//Stop all fade effects for camera "global._camera"
ex_camera_stop_fade(global._camera);

Minimum Supported Version

exCamera 0.5.0
