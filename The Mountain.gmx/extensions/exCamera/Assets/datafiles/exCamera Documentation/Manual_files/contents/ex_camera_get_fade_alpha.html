ex_camera_get_fade_alpha

Returns the alpha for the fade effects

Syntax

ex_camera_get_fade_alpha(cameraHandle)
Argument Description
cameraHandle The camera handle
Returns the fade alpha value

Description

This function returns the alpha value for all fade effect functions.

Example

//Return the alpha value for the camera object's fade effects and store it in variable "_value"
_value = ex_camera_get_fade_alpha(global._camera);

Minimum Supported Version

exCamera 0.5.0
