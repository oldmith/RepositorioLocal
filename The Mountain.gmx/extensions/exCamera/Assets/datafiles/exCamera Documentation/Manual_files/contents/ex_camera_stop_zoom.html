ex_camera_stop_zoom

Stops all zoom effects for the camera

Syntax

ex_camera_stop_zoom(cameraHandle)
Argument Description
cameraHandle The camera handle
Returns nothing

Description

This function will stop any zoom effects currently playing from the particular camera.

Example

//Stop all zoom effects for camera "global._camera"
ex_camera_stop_zoom(global._camera);

Minimum Supported Version

exCamera 0.5.0
