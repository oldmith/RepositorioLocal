ex_camera_shake

Shakes the camera

Syntax

ex_camera_shake(cameraHandle, shakeX, shakeY, shakeAngle, duration)
Argument Description
cameraHandle The camera handle
shakeX The shake X value
shakeY The shake Y value
shakeAngle The shake angle value, in degrees
duration The duration of the effect, in steps
Returns nothing

Description

This function will perform a shake effect.

Example

//Shake the camera view for 20 steps
ex_camera_shake(global._camera, 5, 5, 2, 20);

Minimum Supported Version

exCamera 0.5.0
